# Support

## Community Support

- **Documentation**: Check the docs/ directory and README.md
- **Issues**: Use GitHub Issues for bug reports and feature requests
- **Discussions**: Use GitHub Discussions for questions and community support

## Before Opening an Issue

1. Check the documentation (docs/ directory, README.md)
2. Search existing issues for duplicates
3. Run `bash scripts/validate.sh` to check your setup
4. Try `opencode models` to verify model availability

## Getting Help

1. Read the troubleshooting guide: docs/troubleshooting.md
2. Search GitHub Issues for similar problems
3. Open a new issue with:
   - Your configuration (sanitized)
   - Exact steps to reproduce
   - Expected vs actual behavior
   - Relevant logs and output
   - OpenCode version and model provider

## Scope

This project provides the agent-loop orchestration framework. For issues with:
- Specific model providers: Contact the provider's support
- OpenCode itself: OpenCode documentation and issues
- Your project's code: Your project's support channels
