# OpenCode Agent Loop v0.2.0

Version 0.2.0 focuses on reliability, portability, and observability.

Highlights:

- persistent token, cost, and workflow-call budgets
- same-model transient retries before provider failover
- provider adapter interface with local/Ollama recognition
- versioned structured event logging and a query utility
- stronger recursive log redaction
- portable checkpoint and executable handling
- Linux, macOS, and Windows runtime validation
- scheduled patched-OpenCode compatibility builds

The parent orchestrator remains paid and bounded by a workflow-call ceiling and a lower maximum turn count. Exact parent-session token and dollar use is not available to the plugin, so budget output identifies that limitation rather than presenting worker totals as complete task cost.
